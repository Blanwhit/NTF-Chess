How to Play

NTF Chess is a variant of chess where altruistic actions are rewarded in-game.

Each piece has the same movement as in standard chess. The game ends when one player captures the other player's king, and the altruism points accumulated by players, which are shown by the Altruism Bar, are counted.


  • The game starts with 0 altruism points for both players.

About the Project

NTF chess is being developed by a group of Non-Trivial finalists, who hoped to create an Effective Altruism-themed chess variant for Non-Trivial members to enjoy.

It was developed by this awesome group, and the game is open-source, with the source code available on Github.
